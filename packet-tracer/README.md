# Packet Tracer Labs

This folder contains Cisco Packet Tracer networking labs.

Each lab is used to practice building, testing, and troubleshooting network configurations.

## Labs

### Bad Gateway Lab

File: `Bad Gateway.pkt`

This lab focuses on diagnosing and fixing connectivity issues related to gateway configuration.

Skills practiced:

- IP addressing
- Gateway configuration
- Connectivity testing
- Basic troubleshooting
