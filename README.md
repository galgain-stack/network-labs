# Network Labs

This repository contains hands-on networking labs built using Cisco Packet Tracer.

The purpose of this project is to practice building, testing, documenting, and troubleshooting network environments using an operations mindset.

---

## Goals

- Build small network environments
- Practice configuration and troubleshooting
- Document network topology and changes
- Track work using Git version control
- Develop operational thinking used in real infrastructure environments

---

## Tools Used

- Cisco Packet Tracer
- Ubuntu Linux
- Git and GitHub
- Terminal-based workflow

---

## Labs

### Packet Tracer Labs

- [Network Foundations Lab](packet-tracer/network-foundations-v1.pkt)
- [Lab Documentation](packet-tracer/README.md)

### Network Foundations Lab

This lab focuses on understanding network connectivity problems and troubleshooting gateway issues.

Skills practiced include:

- IP addressing
- Routing concepts
- Connectivity testing
- Network troubleshooting
- Operational documentation

---

## Project Approach

Each lab in this repository will include:

- Network topology
- Packet Tracer lab file
- Configuration notes
- Validation steps
- Troubleshooting scenarios

This approach reflects real-world operational practices used in infrastructure and network operations roles.
